#### With furniture
Clone gazebo models
```
cd /tmp
git clone https://github.com/willie5588912/gazebo_models.git
mv gazebo_models/* ~/.gazebo/models
```
```
roslaunch iceira_gazebo iceira_all_furniture.launch
```
